# DAO message BBS
1. module install
```
$  yarn install
```

2. Start ganache（GUI and port 7545）


3. Compile and migrate（deploy）
```
truffle compile
truffle migrate
```

4. Build Javascript and CSS
```
$  yarn run webpack
```


5. Start local server
```
$ yarn run webpack-dev-server

or

$ yarn run webpack-dev-server --config webpack.development.config.js
```
